## Présentation

**AZIMUT** est une application Android de conversion bidirectionnelle de coordonnées géographiques entre les formats **GPS (WGS84)** et **MGRS (Military Grid Reference System)**. Conçue pour un usage terrain militaire, outdoor et topographique, elle fonctionne **entièrement hors ligne**.

---

## Fonctionnalités

| Catégorie | Fonctionnalité |
|---|---|
| Conversion | GPS → MGRS · MGRS → GPS · détection auto du format |
| Carte | OpenStreetMap offline · zoom · rotation 2 doigts |
| Navigation | Boussole interactive · recentrage au tap · plein écran |
| Terrain | Appui long → pin bleu avec GPS + MGRS du point |
| Localisation | GPS temps réel → conversion automatique |
| UX | Thème sombre/clair · splash screen · copie presse-papiers |
